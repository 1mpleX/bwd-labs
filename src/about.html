О приложении

Это простое веб-приложение для управления задачами. Оно позволяет создавать, редактировать и удалять задачи, отслеживать их статус, а также организовывать их в проекты.

Приложение было создано для повышения личной продуктивности, удобства и упрощения процесса работы с задачами.

Основные функции:

  • Создание и удаление задач.
  • Изменение статуса задач (в работе или выполнено).
  • Организация задач по проектам.
  • Просмотр списка задач и фильтрация их по статусу.

Технологии, использованные в приложении:

  • HTML5, CSS3 для вёрстки и стилизации.
  • JavaScript для динамических элементов и логики.
  • Сохранение данных в браузере (например, через LocalStorage) для обеспечения работы без серверной части.
